# Movie App
영화소개 웹사이트

## 기능
- 리액트 기본기( JSX & PROPS & STATE )
- API 활용
- 다운로드 횟수 인기순으로 정렬
- 영화 제목, 포스터이미지, 장르, 시놉시스 소개
- 리액트 네비게이션 ( react-router-dom )

## 사용 기술
- Front-End : React JS
- Back-End : 없음
- Database : 없음
- Deployed : github.io

## Start
1. 레포지토리의 소스코드 pull
2. npm i
4. npm start
5. localhost:8080에서 확인
